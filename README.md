# EPA Jumper Single Leg Digital Twin

A ROS2-enabled digital twin framework for an EPA Jumper single-leg robot using MuJoCo simulation.

## Features

- Actual Leg Simulation
- Predicted Leg Simulation
- Real-Time Drift Estimation
- ROS2 Integration
- RViz Visualization
- CSV Logging
- Drift Plotting

## Experimental Results
